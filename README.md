# web.instata

Turn your plain old tabular data (POTD) into Web data with web.instata: it takes CSV as input and generates a HTML document with the data items marked up with  [Schema.org](http://schema.org) terms.


	                       +--------------------+
	+-------+              |                    |            +--------------+
	|  CSV  |              |                    |            |              |
	|-------|              |                    |            |   HTML5      |
	|       | +----------->|     web.instata    |+---------> |              |
	|       |              |                    |            |   Schema.org |
	|       |              |                    |            |              |
	|       |              |                    |            +--------------+
	+-------+              +--------------------+
